## Install
```bash
git clone https://github.com/gxchain/gxchain-alpha.git
cd gxchain-alpha
npm install
```

## Dev
```bash
npm run dev
```

## Build
powered by electron builder
```bash
// build all platform
npm run build

// build mac
npm run build:mac

// build windows
npm run build:win

// build linux
npm run build:linux
```

you could find executable or installer in `build` directory
